# Objetivos do SOE-CCG

## Objetivo Principal

Registrar conhecimento gastronômico de forma estruturada, reutilizável, rastreável, versionada, permanente e independente de tecnologia.

---

## Objetivos Específicos

- Registrar receitas como conhecimento, não como documentos estáticos.
- Registrar execuções separadamente das receitas, preservando o histórico de cada preparo.
- Catalogar ingredientes, técnicas e equipamentos como entidades reutilizáveis.
- Registrar observações e experimentos como parte do processo de evolução do conhecimento.
- Permitir relacionamento entre entidades por meio de identificadores permanentes.
- Preservar todas as versões de todos os registros.
- Garantir que o conhecimento sobreviva a qualquer mudança tecnológica.

---

## O que o SOE-CCG não é

- Não é um gerenciador de receitas.
- Não é um sistema dependente de banco de dados.
- Não é uma interface de consulta.

O SOE-CCG é um motor de conhecimento gastronômico.

---

## Visão de Longo Prazo

Tornar possível responder perguntas como:

- Quais técnicas aparecem com maior frequência nas execuções mais bem avaliadas?
- Quais ingredientes participam de determinadas categorias?
- Quais experimentos originaram determinada receita?
- Quais execuções produziram os melhores resultados para uma receita específica?
