# Nomenclatura

Define as convenções de nomenclatura para todos os artefatos do SOE-CCG.

---

## Arquivos Markdown

Minúsculas, sem acentos, com hífens. Sem espaços.

```
doce-de-leite-artesanal-v1.md
esquema-receita-v1.md
receita-v1.md
```

## Diretórios

Minúsculas, sem acentos, com underline para separação funcional e hífens para sequência.

```
dados/
banco_de_dados/
docs/01-dominio/
docs/04-padroes/
```

## Campos de Metadados (frontmatter)

Minúsculas, com hífens.

```yaml
schema-version: 1
criado-em: 2026-06-25
atualizado-em: 2026-06-25
entidade-id: REC-000001
```

## Colunas SQLite

Minúsculas, com underline.

```
schema_version
criado_em
receita_id
```

## Identificadores de Registros

Prefixo maiúsculo, hífen, número com 6 dígitos.

```
REC-000001
ING-000042
```

## Tags

Minúsculas, sem acentos, com hífens.

```
sem-gluten
alta-precisao
confeitaria-francesa
```

---

## Regra Geral

Nenhum artefato do sistema usa espaços, acentos ou caracteres especiais em seus nomes.
