```
1 conceito → 1 documento oficial → N referências
```

Nunca:

```

```

## Regra de Atualização

Ao criar um novo documento normativo:

1. Adicionar entrada neste índice
2. Retirar qualquer documento anterior que cubria o mesmo conceito
3. Mover o documento anterior para `docs/99-referencias/archive/`
4. Registrar a mudança no CHANGELOG
