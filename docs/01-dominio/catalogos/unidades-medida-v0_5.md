# Unidades de Medida

Catálogo controlado de unidades de medida aceitas no SOE-CCG.

---

## Massa

| Código | Nome        |
|--------|-------------|
| `g`    | grama       |
| `kg`   | quilograma  |
| `mg`   | miligrama   |

## Volume

| Código | Nome        |
|--------|-------------|
| `ml`   | mililitro   |
| `l`    | litro       |
| `cl`   | centilitro  |

## Medidas Culinárias

| Código  | Nome              | Equivalência aproximada |
|---------|-------------------|--------------------------|
| `col-s` | colher de sopa    | 15 ml                    |
| `col-c` | colher de chá     | 5 ml                     |
| `xic`   | xícara            | 240 ml                   |
| `pitada`| pitada            | ~0,3 g                   |

## Unidade

| Código  | Nome        |
|---------|-------------|
| `un`    | unidade     |
| `fatio` | fatia       |
| `ramo`  | ramo        |
| `folha` | folha       |
| `dente` | dente       |

## Temperatura

| Código | Nome       |
|--------|------------|
| `°C`   | Celsius    |

## Tempo

| Código | Nome    |
|--------|---------|
| `s`    | segundo |
| `min`  | minuto  |
| `h`    | hora    |

---

## Regras

- Usar sempre o código da tabela nos registros.
- Novas unidades devem ser adicionadas aqui antes de uso.
- Unidades obsoletas marcadas como `[descontinuada]`, nunca removidas.
