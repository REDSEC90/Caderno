# Tipos de Técnicas

Catálogo controlado de classificações de técnicas no SOE-CCG.

---

## Tipos

| Código                    | Descrição                                          |
|---------------------------|----------------------------------------------------|
| `cozimento-umido`         | Cozimento em líquido: fervura, pochê, vapor        |
| `cozimento-seco`          | Assar, grelhar, tostar, refogar                    |
| `cozimento-em-gordura`    | Fritar, saltear, confitar                          |
| `cozimento-misto`         | Braisagem, estufado, guisado                       |
| `fermentacao`             | Fermentação alcoólica, láctica, natural            |
| `emulsificacao`           | Criação e estabilização de emulsões                |
| `gelificacao`             | Uso de gelatina, ágar, pectina                     |
| `reducao`                 | Concentração por evaporação                        |
| `corte`                   | Julienne, brunoise, chiffonade e similares         |
| `moldagem-e-modelagem`    | Formato e apresentação de preparações              |
| `tempero-e-marinada`      | Salga, marinada, cura a seco                       |
| `conservacao`             | Defumação, salga, pasteurização, esterilização     |
| `confeitaria-e-patisserie`| Massas, cremes, coberturas, açúcar trabalhado      |
| `panificacao`             | Massas fermentadas, modelagem, cocção de pães      |
| `outros`                  | Técnicas sem classificação específica              |

---

## Regras

- Uma técnica possui exatamente um tipo principal.
- Novos tipos devem ser adicionados aqui antes de uso.
- Tipos obsoletos marcados como `[descontinuado]`, nunca removidos.
