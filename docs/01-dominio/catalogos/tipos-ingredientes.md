# Tipos de Ingredientes

Catálogo controlado de classificações de ingredientes no SOE-CCG.

---

## Tipos

| Código                  | Descrição                                      |
|-------------------------|------------------------------------------------|
| `laticinios`            | Leite e derivados                              |
| `carnes-bovinas`        | Cortes e derivados bovinos                     |
| `carnes-suinas`         | Cortes e derivados suínos                      |
| `aves`                  | Frango, pato, peru e similares                 |
| `peixes`                | Peixes de água doce e salgada                  |
| `frutos-do-mar`         | Camarão, lula, mariscos e similares            |
| `hortalicas-folhosas`   | Alface, espinafre, rúcula e similares          |
| `hortalicas-legumes`    | Abobrinha, berinjela, cenoura e similares      |
| `frutas`                | Frutas frescas e secas                         |
| `graos-e-leguminosas`   | Feijão, grão-de-bico, lentilha e similares     |
| `cereais`               | Arroz, aveia, cevada e similares               |
| `farinhas`              | Farinha de trigo, milho, amêndoa e similares   |
| `acucares`              | Açúcar refinado, demerara, mascavo e similares |
| `gorduras`              | Manteiga, azeite, óleo e similares             |
| `temperos-frescos`      | Alho, cebola, ervas frescas                    |
| `especiarias`           | Canela, pimenta, cominho e similares           |
| `fermentos`             | Fermento biológico, químico e similares        |
| `liquidos`              | Água, caldo, vinho, vinagre e similares        |
| `conservas`             | Enlatados, conservas em salmoura e similares   |
| `outros`                | Ingredientes sem classificação específica      |

---

## Regras

- Um ingrediente possui exatamente um tipo principal.
- Novos tipos devem ser adicionados aqui antes de uso.
- Tipos obsoletos marcados como `[descontinuado]`, nunca removidos.
