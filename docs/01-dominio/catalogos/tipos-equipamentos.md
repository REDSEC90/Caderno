# Tipos de Equipamentos

Catálogo controlado de classificações de equipamentos no SOE-CCG.

---

## Tipos

| Código                  | Descrição                                              |
|-------------------------|--------------------------------------------------------|
| `medicao`               | Balança, termômetro, cronômetro, densímetro            |
| `coccao-direta`         | Fogão, forno, chapa, brasa                             |
| `coccao-indireta`       | Banho-maria, sous vide, defumador                      |
| `refrigeracao`          | Geladeira, freezer, abatedor de temperatura            |
| `processamento`         | Liquidificador, processador, mixer, batedeira          |
| `corte`                 | Faca, mandoline, cortador, ralador                     |
| `formas-e-moldes`       | Formas de bolo, assadeiras, aros, moldes de silicone   |
| `armazenamento`         | Potes, sacos a vácuo, recipientes herméticos           |
| `utensilios-de-mao`     | Fouet, espátula, colher, pincel, peneira               |
| `panelas-e-recipientes` | Panela, caçarola, frigideira, wok                      |
| `fermentacao`           | Pote de fermentação, airlock, câmara de fermentação    |
| `outros`                | Equipamentos sem classificação específica              |

---

## Regras

- Um equipamento possui exatamente um tipo principal.
- Novos tipos devem ser adicionados aqui antes de uso.
- Tipos obsoletos marcados como `[descontinuado]`, nunca removidos.
